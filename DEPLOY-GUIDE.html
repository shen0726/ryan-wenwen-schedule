🚀 PWA 部署指南

将你的日程表应用部署到互联网

⚡ 推荐：一键自动部署

在终端中运行以下命令，自动完成所有步骤

cd "/Users/rshen/Desktop/Claude code"
./deploy.sh
1

创建 GitHub 仓库

GitHub 是代码托管平台，Vercel 会从这里获取代码进行部署。

  • 访问 github.com/new
  • Repository name 填写: ryan-wenwen-schedule
  • 选择 Public（公开）
  • 不要勾选 "Add a README file"
  • 点击 Create repository
💡 创建后，GitHub 会显示一个页面，里面有下一步的命令。复制 "…or push an existing repository" 部分的代码。
2

推送代码到 GitHub

在终端中运行以下命令（将 YOUR_USERNAME 替换为你的 GitHub 用户名）：

# 进入项目目录
cd "/Users/rshen/Desktop/Claude code"

# 添加远程仓库
git remote add origin https://github.com/YOUR_USERNAME/ryan-wenwen-schedule.git

# 推送代码
git push -u origin main
💡 第一次推送会要求你登录 GitHub，按提示操作即可。
3

部署到 Vercel

Vercel 会自动部署 GitHub 仓库中的代码，并提供免费域名。

  • 访问 vercel.com/new
  • 点击 Continue with GitHub 登录
  • 找到并选择 ryan-wenwen-schedule 仓库
  • Framework Preset 选择 Other
  • Build Command 留空
  • Output Directory 填写 .（一个点）
  • 点击 Deploy
⚠️ 注意：Output Directory 必须是 .（点），因为我们的 index.html 在根目录。
4

添加到手机主屏幕

部署完成后，Vercel 会给你类似 ryan-wenwen-schedule.vercel.app 的网址。

iPhone/iPad (Safari):

  • 用 Safari 打开部署后的网址
  • 点击底部中间的分享按钮 ⬆️
  • 选择 "添加到主屏幕"
  • 点击 "添加"

Android (Chrome):

  • 用 Chrome 打开部署后的网址
  • 点击右上角菜单 ⋮
  • 选择 "添加到主屏幕"
  • 点击 "添加"
🎉 完成！你的应用现在会像原生 App 一样，有启动图标、全屏显示、离线可用！

📋 验证清单

  • 应用能在浏览器正常打开
  • 有自定义图标（RW 紫色图标）
  • 添加到主屏幕后显示正常
  • 离线刷新页面能正常使用（测试：断网后刷新）
🧪 先本地测试 🚀 开始部署

遇到问题？检查以下常见原因：

• 图标未显示 → 检查 icons/ 目录是否在根目录
• 离线不工作 → 检查 service-worker.js 是否在根目录
• 部署失败 → 确认 Output Directory 设置为 .
